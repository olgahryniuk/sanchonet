# Using the faucet

## Request funds from the faucet

1. Go to https://faucet.sanchonet.world.dev.cardano.org/basic-faucet
2. Tick the reCaptcha box 
3. Paste your address from the SanchoNet on the corresponding box:
4. Click on Submit

![](../images/requestFunds.png)


## Request a stake delegation for your stake pool

1. Go to https://faucet.sanchonet.world.dev.cardano.org/basic-faucet
2. Tick the reCaptcha box 
3. Paste your SanchoNet Stake Pool ID on the corresponding box:
4. Click on Submit

![](../images/requestDelegation.png)
